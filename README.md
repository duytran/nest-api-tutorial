## Installation

```bash
$ yarn
```

## Running the app

```bash
# start postgres in docker and push migrations
$ yarn db:dev:restart

# start api in dev mode
$ yarn start:dev
```

## Test

```bash
# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
